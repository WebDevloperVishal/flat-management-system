# Flat Management System (Full-Stack Project)

A **complete full-stack real-estate flat management system** built with **React, Node.js, Express, Prisma, and MySQL**.
This project demonstrates a **real-world CRUD application** with authentication, admin dashboards, image uploads, and user role management.

---

## Features

### **User Roles**

* **Buyer/Seller**

  * Register/Login with secure JWT authentication
  * Add flats with images and details
  * View approved flats and send enquiries

* **Admin**

  * Approve or reject pending flats
  * View sold flats with buyer information
  * Manage all enquiries and users

### **Flat Management**

* Submit new flats with multiple images
* Approve, reject, or mark flats as sold
* Track sold flats with buyer info

### **Enquiry System**

* Buyers can send enquiries for flats
* Sellers can view enquiries for their flats
* Admin can see all enquiries

### **Other Features**

* Role-based route protection with JWT
* Image uploads using **Multer**
* Clean **React frontend** with nested routing
* **Dashboard layout** for users and admin
* Sorting and filtering of flats by status

## Usage

1. Open frontend (React) in your browser.
2. Register as a new user or login as admin.
3. Add flats, view approved flats, send/receive enquiries.
4. Admin can approve/reject flats and view all sold flats.
